# Raw Fonts Overview
This directory contains the "raw" fonts.  These are fonts downloaded from various 3rd party sources, which are processed using FontForge and HarfBuzz into the final files.

# Font Sources

## Default Fonts

- Carlito
	- Source: Google Fonts
		- https://fonts.google.com/specimen/Carlito
- Century (C059)
	- Source: URW Base 35 Fonts
- Courier (Nimbus Mono)
	- Source: URW Base 35 Fonts
- Garamond
	- Source: Google Fonts
		- https://fonts.google.com/specimen/EB+Garamond
- Gothic
	- Source: URW Base 35 Fonts
- Nimbus Sans
	- Source: URW Base 35 Fonts
- Palatino (P052)
	- Source: URW Base 35 Fonts
- Times (Nimbus Roman)
	- Source: URW Base 35 Fonts
- Dingbats (`D050000L.otf`)
	- Source: URW Base 35 Fonts
	- Not built by `dev/generate_fonts.sh`. The shipped `fonts/Dingbats.woff` was produced outside of this pipeline.
