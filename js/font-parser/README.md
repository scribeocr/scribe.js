# font-parser
This repo contains code used by scribe.js for reading, writing, and modifying fonts.  This repo should be considered part of scribe.js for licensing purposes.

This repo started as a fork as [opentype.js](https://github.com/opentypejs/opentype.js), but has drifted away from that project over time.  Opentype.js is MIT licensed.  I am happy to contribute any changes/features back to opentype.js if there is interest from that project’s maintainers.
